* `C99` 语言标准

#### 介绍

在 `Linux C` 中, 我们可以使用 `BSD Socket Api` 来创建套接字, 只是操作底层接口太过与繁琐, 所有就有了 `libsock`, 它可以帮助开发者简化在 `Linux C` 中创建套接字的操作.

具体的使用方式可以查看 `test/*.c`.
